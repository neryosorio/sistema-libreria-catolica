# Sistema de Gestión para Librería Católica

## Descripción

Sistema web desarrollado para administrar de forma integral una librería católica, permitiendo gestionar inventario, clientes, vendedores, ventas, créditos, devoluciones, contratos digitalizados, comisiones y clientes problema.

## Objetivo

Desarrollar una plataforma web que permita optimizar los procesos administrativos y comerciales de una librería católica, facilitando el control de inventario, ventas a crédito, seguimiento de clientes morosos y gestión de comisiones para vendedores.

## Funcionalidades Principales

- Gestión de usuarios y roles
- Gestión de clientes
- Gestión de vendedores
- Control de inventario
- Registro de ventas
- Ventas al contado y crédito
- Gestión de contratos digitalizados
- Gestión de devoluciones
- Control de clientes problema
- Seguimiento de visitas de cobranza
- Gestión de comisiones
- Generación de RVI y vouchers
- Reportes administrativos
- Sistema de alertas
